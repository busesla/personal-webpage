2. Shopping Mall Event Data Collection and API Integration

This project utilized BeautifulSoup (bs4) for web scraping to collect standardized event data (name, date, location, etc.) from shopping malls across Turkey's three major cities (Ankara, Istanbul, and Izmir). We implemented site-specific parsing logic to accommodate diverse HTML structures, unifying the outputs into a common schema. A central main.py script aggregated the data, which was then made accessible through an API. This process established a robust data pipeline, allowing external systems to easily consume the standardized event information.

Technologies Used: QGIS, Python, MapInfo, etc.
